# Recursos Globals

`logo_mountain_runners.jpeg` és el logotip oficial de l'Associació Esportiva
Mountain Runners del Berguedà. Una persona mantenidora del projecte el va
proporcionar i aprovar per a la publicació el 19 de juliol de 2026.

El logotip és un recurs de marca de l'associació, conserva els seus drets i no
queda cobert per la llicència MIT del programari. Qualsevol reutilització fora
d'aquest projecte requereix autorització de l'associació.
